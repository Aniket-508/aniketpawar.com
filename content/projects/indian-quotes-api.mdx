## Overview

REST API for inspirational quotes from Indian founders, designed for demos, bots, and side projects.

## The problem

Generic quote APIs rarely focus on regional entrepreneurs and consistent attribution.

## Wins

Simple endpoints, typed responses, and a public playground on Vercel.

## Tech stack

Next.js route handlers with edge-friendly JSON responses.

## What's next

Categories by industry and multilingual support.
