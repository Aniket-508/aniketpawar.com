## Overview

Better OG streamlines creating Open Graph images for blogs, products, and docs without hand-maintaining PNGs for every route.

Composable templates keep typography, branding, and layout consistent across pages.

## The problem

OG images are easy to neglect until link previews look broken on Twitter, Slack, or iMessage. One-off designs do not scale when you ship many pages.

## Wins

Reusable template API that teams can adopt without adopting a full CMS.

Type-safe props for titles, descriptions, and brand marks.

## Tech stack

Built with Satori for JSX-to-SVG rendering and Takumi for layout primitives, packaged for Next.js route handlers and edge runtimes.

## What's next

CLI scaffolding and a small gallery of starter themes.

Animated OG variants for launch pages.
