## Overview

Free, open-source presentation components for React built on reveal.js with shadcn/ui-compatible registry setup.

## The problem

Slide decks in code often lack reusable blocks, transitions, and presenter-friendly navigation.

## Wins

Pre-built decks, reveal.js-powered navigation, Shiki code slides, and composable presentation blocks.
