## Overview

Peerlist Publish is a Chrome extension that cross-publishes articles from Peerlist to DEV.to, Hashnode, Medium, Ghost, and more — write once, publish everywhere.

## The problem

Cross-posting the same article to multiple platforms means repeating uploads, API setup, and draft management on every site. There is no single flow that keeps Peerlist as the source of truth.

## Wins

One-click publish or save-as-draft across connected platforms from the article you already wrote on Peerlist.

Google OAuth sign-in with minimal permissions, passed Chrome Web Store review on the first submission.

API keys encrypted at rest with user-controlled deletion for full key lifecycle management.
