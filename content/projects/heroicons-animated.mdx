## Overview

Drop-in animated variants of Heroicons with sensible defaults for hover, tap, and loop states.

## The problem

Teams want motion in UI without rebuilding SVG animations icon by icon.

## Wins

Shipped 316 icons with consistent animation APIs and a browsable docs site.

## Tech stack

React components powered by Framer Motion and TypeScript.

## What's next

Additional icon sets and theme-aware stroke animations.
