## Overview

Free, open-source email components for React built on React Email with shadcn/ui-compatible registry setup.

## The problem

Production-ready email layouts need theme consistency and client-safe markup across many providers.

## Wins

Theme-aware blocks inspired by Vercel, Linear, Stripe, and more, with composable React Email v6 components.
