## Overview

Starter for teams shipping a marketing site and Chrome extension from one repo with aligned tooling.

## The problem

Extension and web apps often diverge in lint rules, UI primitives, and release processes when split across repos.

## Wins

Shared packages for UI and config cut duplicate setup time for new products.

## Tech stack

Turborepo orchestration, Next.js 15, WXT for extensions, shadcn/ui components.

## What's next

Plasmo alternative preset and GitHub Actions release templates.
